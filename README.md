Number Guessing Game - Python Tkinter
This is Project 5 from my Python internship. A simple number guessing game with a GUI made using Tkinter.

What it does
The program picks a random number between 1 and 100. The user has to guess it within 7 attempts. After every guess it tells you if the number is higher or lower than your guess.

How to run
Python 3 should be installed. Tkinter comes with Python so nothing extra to install.

python number_guessing_game.py
Features
Random number is generated every new game
7 attempts per round
Shows if guess is too high or too low
Shows how many attempts are left
Checks for invalid input (letters, out of range numbers)
Press Enter or click the button to submit guess
Play Again option without closing the window
Concepts used
Tkinter for GUI
random module for generating numbers
Event handling (button click, Enter key)
Input validation
Basic game logic
How to play
Run the file
Type a number between 1 and 100
Click Guess or press Enter
Use the hints to narrow down your answer
You have 7 chances total

Python 3
